## 📋 1. PROJECT METADATA

*   **Title:** BrandAgent
*   **Tagline:** The World's First Autonomous AI Social Agency on Arc L1.
*   **Description (Short):** A multi-agent marketing engine that automates social media workflows with agent-to-agent USDC micro-payments via Circle Nanopayments.
*   **Description (Long):** 
    BrandAgent is an autonomous orchestration framework designed for the agentic economy. It replaces the traditional marketing agency with a high-frequency network of AI agents that hire, pay, and verify each other's work.
    
    The system is powered by:
    - **Intelligence:** Google Gemini 1.5 Pro & Flash.
    - **Commerce:** Circle Nanopayments (USDC).
    - **Settlement:** Arc L1 Blockchain.
    
    Every campaign cycle, the **Orchestrator Agent** manages a task-based payroll, settling 14+ on-chain transactions for sub-cent tasks (e.g., $0.001 per verification). This is the "Machine-to-Machine" future of commerce, where human intervention is zero and cryptographic proof of work is absolute.

## 📝 4. CIRCLE PRODUCT FEEDBACK (Required)

**Products Used:** Arc (L1), USDC, Circle Programmable Wallets, Circle Nanopayments.

*   **Choice Rationale:** We needed a stack that supports "high-velocity micro-payments." Arc's USDC-native gas is a game-changer for agent UX.
*   **What Worked:** The REST API for Programmable Wallets is developer-friendly. Creating 6 agent wallets took seconds.
*   **Pain Points:** Testing "Transfer" logic on testnet requires manual faucet funding for the Orchestrator. An "Auto-Funded Sandbox" would speed up dev.
*   **Recommendations:** Provide a "Web3 Agent SDK" that handles both Gemini reasoning and Circle payment triggers in a single lifecycle.
